User Guide

Getting Started

When you load the website, you'll see a few key navigation tools to help you explore the shelter map and site directory.

Left Sidebar Navigation

On the left-hand side, you’ll find two icons:

  • Menu Button (three bars): Click this to access key pages:
    • Home – where the interactive map is located
    • User Guide – the page you're reading now
    • SHARE/WHEEL Website – links to the official homepage
  • Map Filter Button: Clicking this opens a tab where you can filter shelter sites by specific features. This helps narrow down the map to only show sites that match your needs and criteria.

Right Sidebar: View & Add Sites

On the right-hand side of the map, you’ll see a sidebar with two main options:

  • Current Shelters: Browse the list of existing shelters. You can search by address using the search bar to quickly locate a site.
  • Add Shelter: Click this to open a form where you can:
    • Search for a location by address
    • Select the shelter type
    • Indicate available utilities and amenities
    • Add optional notes

After submitting the form, you’ll see a confirmation message indicating whether your shelter was:

  • Successfully submitted
  • Failed to submit (likely because a shelter already exists at that location)

Viewing Your Submitted Site

To view your newly added shelter, return to the Current Shelters tab and use the search bar to find it by address. The details you entered will appear with the shelter, including any notes shown at the bottom.

As more shelters are added, use the search bar for quicker access. Selecting a shelter will automatically zoom the map to its location.
